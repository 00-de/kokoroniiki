# 🌸 こころ日記 DX+

シニア向けやさしい日記・予定管理アプリです。

## 主な機能

* 大きな文字表示
* 今日の予定管理
* 音声読み上げ
* 緊急連絡先
* お薬チェック
* 季節背景変更
* 天気表示
* データ自動保存
